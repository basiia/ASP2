## Instrukcja uruchomienia projektu

### Uruchamianie z poziomu Visual Studio:
1. Otworzyć  w Visual Studio UniDesk.sln
2. Zbudować projekt - Build → Build Solution
3. Uruchomić aplikację klikając na przycisk Start

### Uruchamianie z poziomu terminala:
1. Otworzyć terminal w folderze z projektem
2. Wpisać polecenie dotnet build
3. Wpisać polecenie dotnet run

## Foldery w projekcie:
- Controllers: Zawiera pliki kontrolerów, które odpowiadają za logikę aplikacji i zwracanie widoków
- Views: Zawiera pliki widoków, które są renderowane przez kontrolery
- wwwroot: Zawiera zasoby statyczne
